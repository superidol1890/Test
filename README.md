# Test

A simple mod for Among Us that uses **[BepInEx](https://github.com/BepInEx/BepInEx)** and **[Reactor](https://github.com/Nuclear-Powered/Reactor)** and demonstrates mod loading.

## Features

* Registers as a Reactor-compatible plugin.
* Logs a success message to the BepInEx console.

## Installation

1. Install **[BepInEx](https://github.com/BepInEx/BepInEx)** and **[Reactor](https://github.com/Nuclear-Powered/Reactor)**.
2. Download the `Test.dll` from the **[Releases](https://github.com/superidol1890/Test/releases)** page.
3. Place the `.dll` into your `Among Us/BepInEx/plugins` folder.
4. Launch the game!
